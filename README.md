
# RentWatch

RentWatch is a beginner-friendly full-stack web application that allows users to submit, browse, and search rental reviews and rent pricing reports.

## Features

- Submit rental reviews
- Search rental reports
- Browse apartment reviews
- Store rental data in SQLite
- Flask backend architecture
- HTML frontend templates

## Tech Stack

- Python
- Flask
- SQLite
- HTML
- CSS

## Run the Application

### Create virtual environment

```bash
python -m venv .venv
```

### Activate virtual environment

Windows:

```bash
.venv\Scripts\activate
```

### Install requirements

```bash
pip install -r requirements.txt
```

### Run app

```bash
python main.py
```

Open:

```text
http://127.0.0.1:5000
```
